# Changelog

## v0.2.0 - 2026-03-03

- Added aviation great-circle arc rendering with segmented route geometry.
- Added aviation segment details panel with per-segment distance and duration.
- Upgraded shipping mode to a local navigable maritime waypoint network approximation.
- Improved map rendering around antimeridian/world-wrap to avoid broken route patterns.
- Added endpoint-to-sea snapping guardrails for inland address inputs.

## v0.1.0 - 2026-03-03

- Initial release of OSRM Cloud UI.
- Added address geocoding mode (Nominatim).
- Added aviation and shipping great-circle modes.
- Added profile guard for public OSRM demo endpoint.
